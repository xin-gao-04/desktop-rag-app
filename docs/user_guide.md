# User Guide

## Start app
1. `npm install`
2. `npm run dev`

## Configure
- Fill Python executable (default `python`).
- Fill RAG workdir (`<project>\rag-core`).
- Set models and DB path.
- Save config.

## Data add/remove
- Enter a markdown root path and click Add.
- Use Remove button for unwanted sources.
- Click Rebuild Index after changes.

## Query
- Input question in Query panel.
- Click Ask.
- Check answer and source list with snippet.
